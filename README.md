# OSDC website
OSDC's website, built on GatsbyJS and ReactJS.
<br>
## [osdc.surge.sh](https://osdc.surge.sh/)

## Setup
`yarn`

`npm install --global gatsby-cli`

## Running in development
`gatsby develop`
